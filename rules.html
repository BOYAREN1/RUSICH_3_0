Правила сервера [РУСИЧ]

Базовые правила Игровые правила Правила для Сквад-Лидеров Правила для CMD Правила для ТЕХ отрядов Споры по технике Администрация

Базовые правила

1.1 — Ваш НИК должен быть легко читаемым и произносимым.

1.2 — Запрещается совершать любые действия в любой форме, нарушающие действующее законодательство РФ.

1.3 — Запрещается использовать стороннее ПО, макросы или модификации файлов игры, а также её уязвимости, направленные на получение преимущества в игре способами, не предусмотренными разработчиком игры.

1.4 — Саундпад на сервере запрещён в начале и конце матча (во время матча отключить по первому требованию).

1.5 — Создание помех в голосовом чате путём воспроизведения бессмысленных звуков и т.п - ЗАПРЕЩЕНО.

1.6 — Запрещается реклама и агитация в любом виде без разрешения администрации сервера.

1.7 — Оскорбления между игроками не приветствуются. Соблюдайте культуру общения.

1.8 — Оскорбление администрации преследуется по законам военного времени.

Игровые правила

2.1 — Запрещается совершать намеренные действия, направленные на нанесение вреда своей команде (умышленный "слив" техники, умышленное нанесение урона игрокам, постройкам и техники своей команды и т.д.) за исключением случаев убийства/уничтожения по приказу командира стороны, либо командира отряда и/или для предотвращения руина игры.

2.2 — Запрещается совершать умышленные действия, направленные на создание помех в игре другим игрокам каким-либо образом.

2.3 — Запрещается слив/получение информации о противоположной команде.

2.4 — Запрещается находится на сервере и бездействовать более 10 минут. Игра без отряда более 5 минут запрещена (автокик).

2.5 — ДРГ не может быть больше 5 человек.

2.6 — Мейнкемп ЗАПРЕЩЕН (не менее 600м от иконки мейна).

Правила для Сквад-Лидеров

3.1 — Командир отряда обязан говорить на русском языке, взаимодействовать с другими командирами и иметь кит командира отряда. Отсутствие микрофона и коммуникации приведет к расформированию отряда.

3.2 — Командир отряда обязан выполнять приказы CMD.

3.3 — Командир отряда может исключить любого игрока отряда без объяснения причин.

3.4 — Запрещается создание отряда без намерения им командовать.

3.5 — Командование отрядом может быть передано другому игроку только по взаимному согласию (если вам перекинули сквад-лидера без вашего согласия - расформируйте отряд).

3.6 — Разрешается использование БМП в ПЕХ отряде (все БТРы, БМП-1, Лав-25, Страйкер с пулеметом).

3.7 — Запрещается использование техники, принадлежащей другому отряду без согласия Командира отряда, владеющего данной техникой.

3.8 — Запрещается закрывать отряд, состоящий только из Командира отряда. Отряд, состоящий только из Командира отряда, может быть закрыт, только с целью использования боевой легкой техники и вертолёта (передвижные миномёты, ПТУР, СПГ, Зенитные установка, РСЗО).

Правила для CMD

4.1 — CMD обязан находится в пехотном отряде, отряде логистической поддержки, или артиллерийском отряде. Запрещается брать CMD стороны в: технический отряд и ДРГ.

4.2 — Командир стороны управляет всеми отрядами и техникой в его распоряжении (кроме случаев, указанных в п. 4.5).

4.3 — Приказы CMD обязательны к выполнению, кроме приказов, нарушающих правила сервера. Командир стороны имеет право отдавать только общие приказы. Командиры отрядов сами решают, как выполнить поставленную задачу.

4.4 — CMD обязан знать механики игры, иметь понимание того, что происходит на поле боя.

4.5 — Если нет желающих играть за CMD стороны, то его может взять любой командир отряда ради абилок.

4.6 — Если CMD некомпетентный, сквадные вправе потребовать его отстранения.

Правила для ТЕХ отрядов

5.1 — Техническому отряду рекомендуется иметь соответствующее название (тех, верт и тд.).

5.2 — Техника, для управления которой требуется кит Crewman, является тяжелой.

5.3 — Технический отряд может использовать только одну тяжелую технику с экипажем, состоящим из командира отряда и минимум одного члена экипажа.

5.4 — Командир технического отряда обязан иметь снаряжение Lead Crewman и находиться в занятой его отрядом технике.

5.5 — Если в технике остался один член отряда, он обязан вернуть технику на мейн.

5.6 — Запрещается использовать вертолёт, без обладания базовых навыков пилотирования и не возможности его посадить без критических повреждений.

5.7 — Управлять вертолетом разрешается только авиационному отряду в количестве до 5 человек, при обязательном наличии командира отряда с китом «Lead Pilot».

5.8 — Боевой вертолет Апач считается за тяжелую технику и требует полного экипажа из Lead Pilot и Pilot (2 человека).

5.9 — Слив техники запрещен.

Споры по технике

6.1 — Во время отчета стартовой фазы клейм техники разрешен тех отрядам с неполным экипажем. Если до конца отчета стартовой фазы осталось 30 секунд, а тех отряд не имеет двух человек, то он должен отдать технику полному отряду по первому их требованию.

6.2 — После окончания стартовой фазы спор по тяжелой техники решается командой !ROLL между командирами отрядов (если командир отряда отказыввает от команды !ROLL и забирает технику, то наказание - бан).

6.3 — Команда !ROLL прописывается после появления техники на мейне, иначе команда может считаться не действительной.

6.4 — Легкая техника достается тому, кто первый ее занял.

6.5 — Реклейм техники без согласия с лидером отряда – запрещен (включая легкую техники).

Администрация сервера

7.1 — Администраторы проекта, находящиеся на игровых серверах не обязаны тотчас отвечать на вашу жалобу.

7.2 — Окончательная трактовка правил сервера и их изменение остаются за Администрацией проекта.

7.3 — Сроки наказания за нарушение и сами нарушения фиксированными не являются и остаются на усмотрение старшей администрацией проекта.

7.4 — Состав Администрации проекта определяет только Администрация проекта.

7.5 — Препятствие Администрации проекта во время рассмотрения конфликтной ситуации является поводом для наказания.

7.6 — Жалобы на Администрацию проекта принимаются только в специальном канале.

7.7 — Любым сбором средств на развитие проекта занимается только Администрация проекта.

7.8 — Оскорбления Администрации сервера могут повлечь за собой санкции, включая блокировку, как в Discord, так и в игре.
